INTRODUCTION
------------

Ubercart Facebook Pixel module make available for use the Facebook Pixel.
This module control the Facebook Pixel's ViewContent, CompleteRegistration, 
AddToCart, InitiateCheckout and Purchase events to understand the actions 
people take on your website.

The AddToCart, InitiateCheckout and Purchase events are based on the Ubercart 
integration.


INSTALLATION AND CONFIGURATION
------------------------------

* Download the module and copy it into your contributed modules folder:
 (for example, your_drupal_path/sites/all/modules)
  and enable it from the modules administration page
 http://drupal.org/node/1897420 for further information

* Create new Facebook Pixel in Ads Manager
   (https://www.facebook.com/ads/manager/data_sources/pixels/).

* - Configuration:
   Browse to the uc_facebook_pixel configuration page 
   (admin/config/system/uc_facebook_pixel)
   Enter your new Pixel ID and chek options.
